# 第二卷细纲：黑石灰市

## 卷目标

陆沉从赤霄矿场进入黑石城，在没有合法商号、没有本钱、矿籍未销的状态下，用账、税、药、护送和赔付建立沉记行。第二卷的核心不是“卖药发财”，而是让底层散修相信沉记行记的账算数。

## 章节推进

### 第11章 押账入城

陆沉以赤霄门案役身份入城，矿籍未销、债务冻结，行动受南税坊限制。他没有急着销籍，而是在灰巷租下旧茶肆，挂出“代记债账”，确定第二卷切入点：先收债务信息，再建立信用。

### 第12章 三张账

许六的黑牙帮债务引出“债主账、债务人账、收债人账”三张账。沉记行开始把死债改活债，司马珩以聚众放贷之名带走陆沉，实则看见南坊灰税价值。

### 第13章 税司的算盘

司马珩给陆沉南坊散修脚行临记牌，允许代办散修活计登记、收支记录和零税票。沉记行获得第一层合法外皮，但也被税司盯上。

### 第14章 丹渣成药

陆沉找到被青炉丹盟逐出的丹徒薛照，对丹渣分级重配，推出引气水粉和止血糊。沉记行卖的不是高阶丹药，而是低价、稳定、可赔付的生存概率。

### 第15章 黑牙帮的刀

黑牙帮砸送药线，陆沉不正面砸场，而是组织欠债散修停工，用霉米粥棚和烧借据箱迫使常黑牙谈判。黑牙帮从收保护费者变成沉记外送护送服务商。

### 第16章 万宝楼少掌柜

裴玄策带三百灵石试图买沉记行活账和未来股权。陆沉反向抬价，最终借七百灵石，保住账房控制权，只给万宝楼信用分级和有限分红入口。

### 第17章 半价丹潮

沉记行扩产并推出带编号、禁忌、赔付的低阶药包，冲击青炉丹坊低端市场。青炉丹坊以假药栽赃，陆沉借封纸暗记、薛照验药和税司少报税案，反使青炉南铺被封十日。

### 第18章 白灰信

烬骨教杀死七名沉记脚商，以“清账”蛊惑欠债者烧账。陆沉公开承认沉记也是一口井，但强调沉记的井壁可见、账可查、死者可赔，并立下烬骨追账榜。

### 第19章 城北烂庙

陆沉追灰线到城北烂庙，白烬用病童和三十块回脉丹设局，逼沉记行暴露“救不起所有人”的边界。陆沉用药童长约给出冷硬但具体的活路，击碎献祭契页一角，并看见命账细线。半片灰线落入陆沉左腕，把黑井残诀、沉记行账册、万宝楼契约、税司铜牌和药童约串到同一条命线中。

### 第20章 沉记开张

沉记行从临记变正记，青炉丹坊被迫降价，黑石城低阶市场形成可选择格局。沈归衡来信告知赤霄门账崩，陆沉带墨祁、赵铜和护队随万宝楼押车返回赤霄山，第二卷结束。

## 连贯性检查

- 主线清楚：矿奴灰账 -> 散修活账 -> 税司零税 -> 丹渣药包 -> 护送服务 -> 万宝楼贷款 -> 烬骨命账 -> 赤霄门债务爆雷。
- 爽点递进有效：每次冲突都不是单纯打赢，而是把对手的暴力、垄断或脏活改写成沉记行可利用的制度部件。
- 第18章到第19章的白烬线与第一卷祭坛线能接上，且把“命账”从伏笔推进为主线机制。
- 第20章回山钩子成立：沉记行刚站稳，赤霄门债务危机让陆沉有了从灰市经营升级到宗门改制的合理入口。

## 后续注意

- 沉记行暂由周砚、薛照、常黑牙和税司驻点共同维持，第三卷回山时要偶尔回收黑石城远程账报，避免沉记行像被主角丢下。
- 烬骨教“无名”线已露出，不宜在第三卷过快解密；更适合作为陆沉改制过程中不断污染债务和贡献点体系的暗线。
- 陆沉已入引气境，后续修炼提升应继续绑定“账、契、命线”，保持与传统闭关升级的差异；[《无相负债经》](../功法/无相负债经.md)只能作为逐步显形的危险传承，不宜过早变成稳定外挂。
