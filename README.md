# 玄幻异大陆小说设定库

## 暂定书名

《我在异界做黑心巨擘》

备选：

- 《异世资本魔君》
- 《穿越后我把宗门做成了垄断巨头》
- 《苍澜逐利录》

## 核心卖点

现代社会的准黑心资本家陆沉，穿越到异大陆后没有觉醒圣母心，也没有走传统英雄路。他用现代商业、组织、合同、舆论、金融和供应链思维，切入一个以宗门、王朝、世家、商会和邪祟共存的玄幻世界。

他不追求爱情，不追求救世名声，只追求活下去、掌控资源、建立秩序，并最终让整个大陆都绕着他的规则运转。

## 基调

- 男频玄幻，异大陆成长争霸。
- 男主无女主，不写恋爱线。
- 主角狡诈、冷静、杀伐果断，有底线但底线很低。
- 主要爽点来自布局、反杀、资源垄断、制度碾压和势力扩张。
- 世界本身残酷，强者压榨弱者，宗门垄断修炼资源，王朝用律法维稳，商会用债务锁人，邪祟用信仰吞噬人心。

## 一句话故事

一个现代资本游戏的准玩家，穿越到修仙异大陆后，把灵石、丹药、功法、矿脉、情报和人命全部变成筹码，最终从底层矿奴一路爬成大陆秩序的制定者。

## 小说简介

封面：[assets/cover-wo-zai-yijie-zuo-heixin-jubo.png](assets/cover-wo-zai-yijie-zuo-heixin-jubo.png)

陆沉死在现代并购案前夜，醒来时成了沧澜大陆赤霄矿场里一个欠债三十块下品灵石的矿奴。

这里没有系统，没有善意，也没有从天而降的公道。宗门用矿籍锁人，监工用鞭子收账，黑井吞掉一批又一批矿奴，死人甚至还会继续在账册上欠债。

陆沉没有选择热血反抗。他先看账，再看人，最后看谁害怕账被翻出来。

从矿奴灰账到黑井妖潮，从私矿暗道到邪教祭坛，他用债务重算撬动人心，用账册逼疯监工，用妖兽和火线为赫连枭铺出死路。第一笔血账收完后，他带着周砚、墨祁等第一批班底离开矿场，前往黑石城。

在黑石城，他从一块旧门板上的“代记债账”起步，把散修债务、税司零税、丹渣药包、黑牙护送和万宝楼贷款连成沉记行。等烬骨教再次以白灰信和命账契页伸手时，陆沉已经不只是会算钱，他开始学会把人的命也写进自己的规则。

当赤霄门债钟敲响，旧宗门的体面被讨债队伍剥干净，陆沉带着沉记行的账和黑石城的债回山。救宗门只是说法，改宗门的账，才是生意。

在这个灵气衰退、资源垄断、强者制定规则的世界里，陆沉要做的不是成为救世主，而是把所有人的命、债、仇、利都写进自己的账本。

旧规矩吃人不吐骨头。

新规矩至少明码标价。

## 主线结构

第一卷：矿奴开局  
陆沉穿越成赤霄矿场的低等矿奴，在宗门监工、妖兽灾变和矿场暴乱中活下来，并用账册、债务和人心控制第一批班底。

第二卷：黑市立身  
陆沉进入边境城池黑石城，建立沉记行，用代记债账、税司牌照、低价药包、黑牙护送和万宝楼贷款重组底层散修信用，同时与青炉丹盟、烬骨教发生正面冲突。

第三卷：宗门上市  
陆沉借壳一个濒临破产的小宗门，将宗门改造成利益共同体，用贡献点、股份、债务、雇佣修士和情报网络重塑宗门制度。

第四卷：诸侯割据  
大陆灵脉衰退，王朝、宗门、世家全面开战。陆沉以商会、雇佣兵和傀儡诸侯切入乱世，开始建立自己的领地。

第四卷正文已落地为《黑石税风》：赤霄新账取得黑石税司一年观察备案，贡献点三分法和外务债凭被限制流通但未被按死；扶风谷急命伤药账、铁鼎宗劫车损耗账、寒鸦坞死契争议账使东陆外务互保预账成形，为第五卷东陆债盟铺路。

第五卷正文已落地为《东陆债盟》：陆沉把东陆外务互保预账推进为债盟雏形，周砚掌总账、墨祁立黑契司、赵铜成护路队；青炉丹盟公验低阶药线，万宝楼收凭争夺定价权，烬骨教烧账夜失败。陆沉在黑井取得《无相负债经》上卷残文，以命契分账术正式入门，境界推进到引气后期。

第六卷正文已落地为《大胤税火》：大胤贡税令引发税火，纪闻霜代表监察司审查陆沉权力来源，温氏药田、司徒氏粮仓、边郡镇抚司和地方世家陆续下场。陆沉将粮票、流民工籍、护路税、药仓粮仓双账和军账整合为黑石外务区，获得一年临时安置权，成为实际领地控制者。

第七卷规划为《南海万宝》：陆沉进入南海信用战，以沉记系债凭、赔付池和护路信用挑战万宝楼商会簿，并通过潮汐账库确认商会契约与天账同源。

第八卷规划为《北境天账》：陆沉进入北境古战场，面对债魂、灰火祭场、无名看门人和天账残库，夺得天账残页读取权，正式提出人间账网计划。

第九卷规划为《大胤封账》：大胤皇廷和户部天册院下场封存黑石外务区副账，要求陆沉交出债凭、军账、民籍和天账残页副本。陆沉以王朝旧军功、旧民籍和旧贡税坏账反制，换取黑石外务区十年保税试行。

第十卷规划为《天门议账》：天门议庭要求陆沉携天账残页赴会受审，仙门强者试图剥离其读取权。陆沉公开大宗灵脉、天劫转嫁和外门牺牲账，提出天劫成本可审计、可分摊、可追偿。

后续天道垄断线：  
陆沉发现所谓天道也是一套古老的资源分配机制。最终目标不再是成仙，而是夺取天道结算权，让所有修行者都必须向他的秩序缴税。
