# 家族索引

## 已关联正文人物

- [沈氏](沈氏.md)：沈归衡背后的赤霄旧账房传承。
- [裴氏](裴氏.md)：裴玄策背后的南海万宝楼商会世家。
- [司马氏](司马氏.md)：司马珩背后的大胤税吏世家。
- [顾氏](顾氏.md)：顾寒川背后的赤霄掌门一系。
- [韩氏](韩氏.md)：韩炽背后的赤霄旧武力家族。
- [秦氏](秦氏.md)：秦闻舟背后的赤霄内门旧派。

## 后续新增

- [温氏](温氏.md)：第六卷药田世家，连接药材、丹盟和低阶药线。
- [司徒氏](司徒氏.md)：第六卷粮仓世家，连接粮账、军需和中州官场。

## 第八卷补充

- [顾氏](顾氏.md)：北境天账中补出顾氏旁支曾牵涉旧军功平账，为第九卷王朝封账和顾寒川压力线埋伏笔。

## 第十一卷补充

第十一卷西荒妖契线暂不新增传统人族家族，重点改为妖庭大族、小族和外帐战骑的血誓关系。若后续需要，可单独建立妖庭大族文档，用于记录妖族内部的大族战利吞并、小族血债和赤砾母族旧账。
